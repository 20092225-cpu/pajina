¿Qué es HTML?

HTML (siglas de HyperText Markup Language) es el lenguaje de marcado estándar que funciona como el esqueleto digital para estructurar y organizar el contenido de cualquier página web. No se considera un lenguaje de programación porque no procesa lógica ni operaciones matemáticas; en su lugar, utiliza un sistema de etiquetas en parejas para indicarle al navegador de internet la función de cada elemento, definiendo con precisión dónde se colocan los títulos, los párrafos, las imágenes y los hipervínculos. En el desarrollo moderno, HTML representa la base estructural cruda que obligatoriamente se complementa con CSS para aportar el diseño visual y con JavaScript para añadir la interactividad..

Las páginas web están formadas por etiquetas que ayudan a organizar toda la información.

Etiquetas Básicas

<HTML> = Inicio del documento

<HEAD> = Encabezado de la página

<TITLE> = Título de la página

<BODY> = Cuerpo de la página
